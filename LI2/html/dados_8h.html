ESTADO* inicializar_estado ( )

Função para inicializar o estado do jogo, colocando todas as peças do tabuleiro como "VAZIO" menos na posição "e5" que fica definida como "BRANCA".

Retorna
O Estado do jogo ja inicializado.

Atribui uma peca "BRANCA" na posição "e5".

Define como ultima jogada a peça inicial, para ser utilizado futuramente na funçao "jogar".

Inicializa o numero de jogadas, atribuindo-lhe o valor '0'.

Coloca como jogador atual o jogador 1, visto que o jogo começa sempre por ele.

Função auxiliar para a função "pos" na camada dados, atualiza o estado do jogo para que esta seja executada com sucesso.

Parâmetros
eEstado atual do jogo.
rAlgarismo que vem a seguir ao comando "pos", ("pos 5", r = 5).

Semelhante ao que acontece na função "*inicializar_estado()", coloca o tabuleiro com peças vazias.

Coloca a peça na casa "e5" como "BRANCA", e define como "ultima jogada" esta casa.

Coloca o numero de jogadas e o jogador atual como 0 e 1, respetivamente.

Usa a função "jogar()" e os dados do array "jogadas" para atualizar o tabuleiro recentemente reinicializado.

Função que coloca todas as jogadas anteriores no devido array na struct.

Parâmetros
eEstado atual do jogo.
cordString de onde vão ser tiradas as coordenadas para serem colocadas como jogadas anteriores.
jogVariável que indica qual das partes das funções será utilizada (1 ou 2).
indexIndice a ser usado para colocar os dados no array.

No caso de a jogada ainda ser do jogador 2, nao ha registo da jogada deste, pois ainda nao foi feita, logo, apenas atualiza a jogada do jogador 1 no array jogadas.

No caso de a jogada ser do jogador 1, entao tem de haver registo de ambos os jogadores, logo, coloca-se as jogadas de ambos os jogadores no array jogadas.
